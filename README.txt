Cara jalankan kode:
1. Pindahkan semua file kedalam folder, setelah itu masukkan folder tersebut ke "C:\xampp\htdocs"
2. Masuk ke aplikasi XAMPP
3. Start Apache dan MySQL
4. Pilih "Admin" di mySQL
5. Pilih Import, dan masukkan file db.sql yang ada di folder
6. Masuk ke Browser dan masukkan URL "http://localhost/(nama folder yang dimasukkan tadi)"
